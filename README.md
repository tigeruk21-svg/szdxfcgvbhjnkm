# MockFort Demo — декоративный репозиторий (фейк)

![status](https://img.shields.io/badge/status-demo-blue) ![license](https://img.shields.io/badge/license-MIT-lightgrey)

Описание
--------
MockFort Demo — это полностью безопасный, образовательный и демонстрационный проект, имитирующий структуру «инструмента» в стиле game‑tool для учебных целей. Проект содержит только заглушки и симуляцию модулей на C++: никакого реального доступа к процессам, памяти, сети или попыток обойти защиту — всё фиктивно.

Важно
------
- Этот репозиторий — исключительно для обучения, демонстрации архитектуры и дизайна интерфейса.
- Запрещено использовать материалы для создания читов, вредоносных программ или нарушения правил сервисов.
- Если вы изучаете безопасность игр — делайте это только в разрешённой среде, с согласия владельца и в рамках закона.

Что внутри
----------
- C++ проект (CMake) с простыми заглушками:
  - src/main.cpp — консольное меню демонстрации
  - src/fake_modules.cpp/.h — имитация «модулей», возвращающих фиктивные данные
  - src/ui_stub.cpp/.h — заглушка интерфейса
- LICENSE (MIT)
- README — это файл

Как собрать
-----------
1. mkdir build && cd build
2. cmake ..
3. cmake --build .

Пример запуска
--------------
$ ./mockfort
Menu:
1) Запустить демонстрацию
2) Показать модули
3) Выход

Примечание по этике
-------------------
Если ваша цель — изучение безопасности или разработка античита — отлично. Поддерживаю ответственное исследование. Если хотите, могу предложить альтернативные образовательные сценарии: тестовый челлендж по защите памяти, шаблон для античит‑модулей (этичный), инструкции по юридическим аспектам исследования безопасности и т.д.
